# 🎬 One-Hand Video Player

한 손으로 편하게 동영상을 시청할 수 있도록 설계한 웹 기반 동영상 플레이어입니다.

## 📖 프로젝트 소개

대부분의 동영상 플레이어는 하단 가로 타임라인을 사용합니다.

하지만 실제로는 다음과 같은 상황에서 동영상을 시청하는 경우가 많습니다.

* 🚇 지하철에서 손잡이를 잡고 있을 때
* 🚌 버스에서 서서 이동할 때
* 🚶 걸어가면서 영상을 볼 때
* 🪥 양치하면서 영상을 볼 때
* 🍔 한 손에 음식을 들고 있을 때

이러한 상황에서는 기존의 가로 타임라인을 조작하기 어렵습니다.

One-Hand Video Player는 **세로 타임라인 UI**를 통해 엄지손가락 하나만으로 동영상 탐색이 가능하도록 설계되었습니다.

---

## ✨ 주요 기능

### 📂 로컬 동영상 재생

* MP4 등 브라우저에서 지원하는 영상 파일 재생
* 파일 업로드 없이 즉시 재생
* PC 및 모바일 브라우저 지원

### ▶️ 유튜브 영상 재생

* 유튜브 링크 입력 지원
* YouTube IFrame API 활용
* 기존 유튜브보다 한손 조작에 최적화된 UI 제공

### 📏 세로 타임라인

* 세로 형태의 재생바 제공
* 위아래 드래그로 탐색 가능
* 엄지손가락 조작에 최적화

### 🤚 왼손 / 오른손 모드

* 버튼 하나로 UI 위치 변경
* 왼손 사용자 지원
* 오른손 사용자 지원

### 📱 모바일 가로 화면 최적화

* 전체 화면 동영상 재생
* 최소한의 UI 구성
* 영상 시청 영역 최대 확보

## 🎯 프로젝트 목표

기존 동영상 플레이어는 "양손 사용"을 전제로 설계되어 있습니다.

본 프로젝트는 다음 질문에서 시작되었습니다.

> "지하철에서 손잡이를 잡고 있는 상태에서도 편하게 동영상을 탐색할 수 있을까?"

이를 위해 기존의 하단 가로 타임라인 대신 세로 타임라인 인터페이스를 실험적으로 적용하였습니다.

## 🚀 사용 방법

### 로컬 영상 재생

1. HTML 파일 실행
2. 영상 파일 선택
3. 재생 버튼 클릭
4. 세로 타임라인으로 탐색

### 유튜브 영상 재생

1. 유튜브 링크 입력
2. 재생 버튼 클릭
3. 세로 타임라인으로 탐색
